# メモ書き

## 前提（スコープ）など
- ジャグラーの設定狙い特化で良い
- Firebase Hosting + WPA
  - バックエンドが必要になったらAWS?

## 機能要件
- 確率計算
  - 回転数・BIG・REGを入れて、それぞれの確率（合算含む）を計算
- 台番メモ
  - 複数のセットで管理できるようにする
    - 「こいつらは凹み台、こいつらは回ってない台」など
- 複数台合算計算
  - 単体
  - 複数台
- 打った台の履歴
  - 開始時ステータス（回転数/BIG/REG）・ヤメ時ステータス・その台の根拠
- データの保存
  - LocalStorage or IndexedDB
  - 「PCと連携して、後でデータ閲覧」は不要？

## 非機能要件
- スマホでサッと入力できるUI/UX
  - 周回しながらポチポチできるように
